Реализация многослойного персептрона с двумя скрытыми слоями.

Были реализованы функции активации/деактивации Sigmoid, ReLU, Tanh. Было реализовано прямое и обратное прохождение,адаптивныцй шаг обучения, L2 регуляризация и функция потерь. Также была реализована преждевременная остановка программы для борьбы с переобучением, если Loss функция будет расти.
